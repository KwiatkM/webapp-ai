# ASP.NET Core MVC WebApp

## Uruchomienie projektu

1. Upewnij się, że masz zainstalowane .NET SDK (https://dotnet.microsoft.com/download)
2. W terminalu przejdź do katalogu projektu.
3. Uruchom polecenie:

    dotnet run

4. Otwórz przeglądarkę i przejdź do adresu wyświetlonego w terminalu (np. http://localhost:5000)

## Struktura projektu
- `Controllers/` – kontrolery aplikacji
- `Views/` – widoki (pliki .cshtml)
- `Models/` – modele danych
- `Startup.cs` – konfiguracja aplikacji
